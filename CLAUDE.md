# VS Code PDF Viewer MCP Server

A stdio-based MCP server that lets Claude read and search PDFs opened in the VS Code PDF viewer extension.

## Build

```bash
npm install
npm run build
```

## Project Structure

- `src/index.ts` - Main server implementation (all tools defined here)
- `build.js` - esbuild bundler script
- `build/index.js` - Bundled output (generated by `npm run build`)

## How It Works

The VS Code PDF viewer extension writes PDF state (text content, selection, metadata) to a temp file (`$TMPDIR/vscode-pdf-viewer-mcp-state.json`). This MCP server reads that file to answer tool calls.

## Workspace Config

The `workspace-config/` folder contains files meant to be copied into target workspaces that use this MCP server:
- `CLAUDE.md` - Instructions for Claude on how to use the PDF tools
- `settings.json` - Claude Code MCP server configuration (goes in `.claude/settings.json`)
